🛒 ProductCategorySystem

A modern ASP.NET Core Web API built with .NET 9 using Clean Architecture Pattern.  

---

📌 About

Backend Web API for managing products and categories.  
Built with Clean Architecture Pattern and Entity Framework Core.
It manages Products and Categories with a Many-to-Many relationship.


---

✨ Features

- 🛒 CRUD for Products  
- 📂 CRUD for Categories  
- 🔗 Many-to-Many relationship  
- 🧱 Clean Architecture Pattern (Domain / Application / Infrastructure / API)  
- 🗄 EF Core + SQL Server  
- 📄 Swagger API  

---
 
🛠 Tech Stack

- ASP.NET Core  
- C#  
- Entity Framework Core  
- SQL Server  

---

🚀 How to Run

1. Install requirements
Make sure you have installed:
- .NET 9 SDK
- SQL Server

2. Clone the repository
